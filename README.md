# Jack Preparacoes Page

Landing page em React/Vite para a barbearia Jackson Preparacoes.

## Tecnologias

- React
- Vite
- TypeScript
- Lucide React
- Google Fonts: Bebas Neue e Barlow

## Rodando localmente

```bash
npm install
npm run dev
```

O projeto abre em `http://127.0.0.1:5173/`.

## Build

```bash
npm run build
```

## Conteudo

A landing apresenta a barbearia, direciona o cliente para o app Android, mostra servicos selecionados, barbeiros, contato, endereco, Instagram e WhatsApp.
